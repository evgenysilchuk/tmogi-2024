**66. Определить среднюю квадратическую погрешность вычисленной длины окружности (в метрах), если её радиус R измерен со средней квадратической погрешностью mR=0.03 м.**  

**Дано:**  
mR=0.03 м  

**Решение:**    
$$P=2\pi R$$  
$m_{P}=2\pi m_{R}$ 

**Ответ:** 
$m_{P}=0.019m$ 

**Решение на C++:**  
```c++
#include <iostream>
#include <math.h>
using namespace std;

int main() {
    double m_R = 0.03,
        m_P;
        
    m_P = 2 * M_PI * m_R;
    m_P = round(m_P * 100) / 100;
    cout << "m_P = " << m_P << "m" << endl;
}
```
